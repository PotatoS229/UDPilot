## 👋 Приветствие от автора

Привет! Меня зовут **Артемий Максалиев**. Я студент Московского энергетического института (МЭИ) и разработчик, который горит идеей создания по-настоящему качественного и доступного VPN-сервиса.

### 💭 Моя мечта
Я верю, что безопасный и быстрый интернет должен быть доступен каждому, независимо от финансовых возможностей. Именно поэтому я начал этот проект — **UDPilot** станет **полностью бесплатным VPN-сервисом корпоративного уровня** с открытым исходным кодом.

### 🎯 Почему именно UDPilot?
Существующие решения либо дороги, либо недостаточно гибки, либо не дружат с UDP-трафиком (читай: с онлайн-играми и VoIP). Я решил объединить лучшее из двух миров:
- **Мощь C++** для низкоуровневой работы с сетью и максимальной производительности
- **Красоту и удобство React** для интуитивного интерфейса
- **Масштабируемость NestJS** для надежной серверной архитектуры

### 🔄 Как это работает?

1. **Пользователь** выбирает сервер и нажимает "Подключиться" в React UI
2. **fastify бэкенд** обрабатывает команду и отправляет ее C++ ядру через IPC
3. **C++ Core**:
   - Поднимает TUN интерфейс
   - При необходимости запускает `udp2raw-tunnel` для оптимизации UDP
   - Собирает метрики скорости и пинга
4. **Визуализация**: метрики в реальном времени отправляются обратно в React и отображаются через D3.js
5. **Кастомные серверы**: пользователь вводит IP/порт — конфигурация сохраняется и используется ядром
